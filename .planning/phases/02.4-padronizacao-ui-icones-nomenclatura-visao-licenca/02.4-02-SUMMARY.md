## Decisions Made

- **Philosophy shift:** List view focused on "what tools does this person have?" rather than "is this person active?"
- **Status/lastLogin on detail only:** Operational data still accessible, just not cluttering the list view
- **Batch fetch:** Single query for all user tool accounts per page, then in-memory mapping (O(n) instead of O(n*m))
- **Deduplication:** If user has 2 Claude accounts, only 1 Claude icon appears
