## Decisions Made
- Used `AIToolType.values()` dynamically for filter options instead of hardcoded string list, ensuring new tool types are automatically available
- Made `DateTimeFormatter` a static constant on the controller for thread safety and efficiency
- Created separate table fragments (toRemoveTable vs externalTable) rather than a single parameterized fragment, since the two sections have different column sets (status vs type, lastSeenAt vs firstSeenAt)
- Included "Usuario" column in the toRemoveTable since those accounts DO have linked users (SUSPENDED/REVOKED accounts had users before being suspended)
