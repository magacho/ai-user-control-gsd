**1. Update GitHubCopilotClient.fetchSeats()** to parse date fields from the seat JSON:
- Import `java.time.Instant`, `java.time.OffsetDateTime`
- In the seat parsing loop (where `login` is extracted from `assignee`), also extract:
  - `String createdAtStr = (String) seat.get("created_at");`
  - `String lastActivityStr = (String) seat.get("last_activity_at");`
- Parse dates using `OffsetDateTime.parse(str).toInstant()` (NOT `Instant.parse()` -- GitHub returns timezone offsets like `-06:00`). Wrap each in null/blank check:
  ```java
  Instant createdAtSource = null;
  if (createdAtStr != null && !createdAtStr.isBlank()) {
      createdAtSource = OffsetDateTime.parse(createdAtStr).toInstant();
  }
  Instant lastActivityAt = null;
  if (lastActivityStr != null && !lastActivityStr.isBlank()) {
      lastActivityAt = OffsetDateTime.parse(lastActivityStr).toInstant();
  }
  ```
- Replace `new ToolAccountInfo(login, null)` with `new ToolAccountInfo(login, null, createdAtSource, lastActivityAt)`
- Wrap date parsing in try-catch per field (log warning on parse failure, set to null, do NOT crash the seat)

**2. Add `lookupUserByGitName(String gitName)` to GoogleWorkspaceService:**
- Returns `Optional<GwsUser>`
- Uses `directory.users().list()` with:
  - `.setDomain(domain)` (from `appProperties.getGoogleWorkspace().getDomain()`)
  - `.setQuery(customSchemaName + ".git_name='" + sanitizeQueryValue(gitName) + "'")` where `customSchemaName` comes from `appProperties.getGoogleWorkspace().getCustomSchemaName()`
  - `.setProjection("full")`
  - `.setMaxResults(2)` (expect 0 or 1; >1 means duplicates, log warning)
- Execute the request. If `response.getUsers()` is null or empty, return `Optional.empty()`
- If >1 user found, log a warning: "Multiple GWS users found with git_name={}, using first match"
- Map first user via existing `mapToGwsUser()` and return wrapped in `Optional.of()`
- On `IOException`, log error and throw `RuntimeException("GWS lookup by git_name failed for " + gitName, e)`
- Add a private `sanitizeQueryValue(String value)` method that escapes single quotes: `return value.replace("'", "\\'");`

**3. Update AccountLinkingService.linkAccounts()** to persist source dates:

  - If `accountInfo.getLastActivityAt() != null`, set it (always overwrite with latest from API -- per 90-day retention, only overwrite with non-null values to preserve last-known)
